# 🚀 Siddiqur Rahman — 3D Developer Portfolio

A modern, animated 3D developer portfolio built with **React**, **Three.js**, **TailwindCSS**, and advanced motion effects. Showcasing skills, projects, and creativity with stunning visuals and smooth user experience.

![Portfolio Screenshot](https://github.com/user-attachments/assets/9b0ed20e-074e-4f2a-81d8-20c9da751e9e)

---

## ✨ Features

- **3D Visuals**: Interactive 3D astronaut and planet scenes powered by [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/getting-started/introduction) and [Drei](https://github.com/pmndrs/drei).
- **Motion Effects**: Smooth transitions and scroll-based animations using [Motion One](https://motion.dev/) and [Framer Motion](https://www.framer.com/motion/).
- **Responsive UI**: Clean, modern, and fully responsive design with [TailwindCSS](https://tailwindcss.com/).
- **Contact Form**: Working contact form with [EmailJS](https://www.emailjs.com/) integration.
- **Showcase**: Animated project previews, testimonials, and timeline of experience.
- **Performance**: Lightning-fast development and builds with [Vite](https://vitejs.dev/).
- **Custom Components**: Orbiting skill icons, parallax backgrounds, draggable cards, and more.

## 🚦 Getting Started

### 1. Clone the Repository

```sh
git clone https://github.com/siddiqur2002/portfolio.git
cd portfolio
```

### 2. Install Dependencies

```sh
npm install
```

### 3. Configure Environment Variables

Create a `.env` file in the root directory and add your [EmailJS](https://www.emailjs.com/) credentials:

```
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

### 4. Run the Development Server

```sh
npm run dev
```

The app will be available at [http://localhost:5173](http://localhost:5173).
